java集合解读

容器大体上分为：数组（固定大小），集合（大小动态变化）
集合大体上可分为：Collection（一个独立元素的序列，即仅有值），Map（一组成对的“键值对”对象，允许你使用键来查找值，即每一个对象均为“键值对”）
集合：
一，Collection：
1，ArrayList：
特点：ArrayList可以动态增长和缩减（所有集合的共性，也是集合相比于数组的最大区别），底层为数组，线程不安全。
动态增长与缩减的原因：该类封装了一个动态再分配的Object[]数组，使每个类对象均含有一个名为capacity的属性，该属性表示并控制数组的长度，比如：当向ArrayList内添加元素时，该属性值便会增加。
与数组的length不同，ArrayList含有size和Capacity两个属性，数组的length属性即为数组定义时的大小，与数组中的对象个数无关，但ArrayList的size为集合内对象的个数，Capacity为集合的存储空间大小。
Capacity为动态增长和缩减的，其增长规律为：int newCapacity=（oldCapacity*3）/2+1;
